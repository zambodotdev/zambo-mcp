# Zambo MCP

**29 AI tools for builders** ‚Äî strategy generation, code audit, lead gen, onchain scoring, credit discovery, identity verification, prompt defense, and more.

**Zero auth. Zero signup. One JSON line.** Works in Claude, Cursor, Windsurf, Cline, and any MCP client.

## Quick Start (10 seconds)

Add this to your MCP client config:

```json
{"mcpServers":{"zambo":{"url":"https://zambo.dev/api/mcp"}}}
```

Or add both Zambo + Axis MCPs together for all 40 tools:

```json
{
  "mcpServers": {
    "zambo": {"url": "https://zambo.dev/api/mcp"},
    "axis": {"url": "https://zambo.dev/api/axis-mcp"}
  }
}
```

### Claude Desktop
1. Open Settings ‚Üí Developer ‚Üí Edit Config
2. Paste the config above
3. Restart Claude ‚Äî 29 tools appear instantly

### Cursor / Windsurf / Cline
Same config ‚Äî paste into your MCP settings and restart.
